Brainrot Blast

Casual Match-3 Puzzle Game with Italian Brainrot Characters

Project Overview

Brainrot Blast is a casual Match-3 puzzle game featuring collectible Italian brainrot meme characters. Players match colorful icons, complete levels, and unlock brainrot pets/badges as rewards.
